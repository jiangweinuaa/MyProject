#### 2. 修改 `updateProductFeatures()` 方法

添加阿里云类目信息参数：

```java
/**
 * 更新商品特征
 */
private void updateProductFeatures(ProductRecognitionRequest request, 
                                    String aliCategoryId, 
                                    String aliCategoryName) {
    try {
        String checkSql = "SELECT COUNT(*) FROM PRODUCT_FEATURES WHERE PLUNO = ?";
        Integer count = jdbcTemplate.queryForObject(checkSql, Integer.class, request.getPluno());
        
        if (count != null && count > 0) {
            // 更新已有记录（包含阿里云类目信息）
            String updateSql = "UPDATE PRODUCT_FEATURES " +
                "SET IMAGE_COUNT = IMAGE_COUNT + 1, " +
                "ALI_CATEGORY_ID = NVL(?, ALI_CATEGORY_ID), " +
                "ALI_CATEGORY_NAME = NVL(?, ALI_CATEGORY_NAME), " +
                "UPDATED_TIME = SYSDATE " +
                "WHERE PLUNO = ?";
            jdbcTemplate.update(updateSql, aliCategoryId, aliCategoryName, request.getPluno());
        } else {
            // 插入新记录（包含阿里云类目信息）
            String featureId = UUID.randomUUID().toString().replace("-", "");
            String insertSql = "INSERT INTO PRODUCT_FEATURES " +
                "(FEATURE_ID, PLUNO, PRODUCT_NAME, CATEGORY, " +
                "ALI_CATEGORY_ID, ALI_CATEGORY_NAME, " +
                "IMAGE_COUNT, CREATED_TIME, UPDATED_TIME) " +
                "VALUES (?, ?, ?, ?, ?, ?, 1, SYSDATE, SYSDATE)";
            jdbcTemplate.update(insertSql, 
                featureId,
                request.getPluno(),
                request.getProductName(),
                request.getCategory(),
                aliCategoryId,
                aliCategoryName
            );
        }
        
    } catch (Exception e) {
        e.printStackTrace();
    }
}
```

#### 3. 修改 `matchLocalProduct()` 方法

优化匹配逻辑，优先使用阿里云类目 ID 匹配：

```java
/**
 * 匹配本地训练库商品
 * @param categoryName 阿里云识别的类目名称
 * @param categoryId 阿里云识别的类目 ID（如果有）
 * @return 匹配的本地 PLUNO，未匹配返回 null
 */
private String matchLocalProduct(String categoryName, String categoryId) {
    if (jdbcTemplate == null) {
        return null;
    }
    
    try {
        // 策略 1: 精确匹配阿里云类目 ID（最优）
        if (categoryId != null && !categoryId.trim().isEmpty()) {
            String sql = "SELECT PLUNO FROM PRODUCT_FEATURES WHERE ALI_CATEGORY_ID = ?";
            List<Map<String, Object>> list = jdbcTemplate.queryForList(sql, categoryId);
            if (list != null && !list.isEmpty()) {
                return (String) list.get(0).get("PLUNO");
            }
        }
        
        // 策略 2: 精确匹配商品名称
        // ... 原有逻辑
        
        // 策略 3: 模糊匹配商品名称
        // ... 原有逻辑
        
        // 策略 4: 匹配阿里云类目名称
        if (categoryName != null && !categoryName.trim().isEmpty()) {
            String sql = "SELECT PLUNO FROM PRODUCT_FEATURES WHERE ALI_CATEGORY_NAME = ?";
            List<Map<String, Object>> list = jdbcTemplate.queryForList(sql, categoryName);
            if (list != null && !list.isEmpty()) {
                return (String) list.get(0).get("PLUNO");
            }
        }
        
        return null;
        
    } catch (Exception e) {
        System.err.println("匹配本地商品失败：" + e.getMessage());
        e.printStackTrace();
        return null;
    }
}
```

## ⚠️ 注意事项

1. **API 响应解析**: 需要从阿里云 API 响应中提取真实的 `CategoryId`
2. **历史数据**: 已有的训练数据没有阿里云类目信息，需要重新训练或手动补充
3. **匹配准确率**: 使用类目 ID 匹配比名称匹配更准确
