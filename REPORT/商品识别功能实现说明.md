### 4. 数据库表

**表名**: `PRODUCT_RECOGNITION_LOGS`

**字段说明**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `LOG_ID` | VARCHAR2(64) | 主键，UUID |
| `IMAGE_URL` | VARCHAR2(1000) | 识别图片 URL |
| `RECOGNIZED_PLUNO` | VARCHAR2(50) | 识别出的品号 |
| `RECOGNIZED_NAME` | VARCHAR2(200) | 识别出的商品名称 |
| `CONFIDENCE` | NUMBER(5,4) | 置信度 (0-1) |
| `USER_CONFIRMED_PLUNO` | VARCHAR2(50) | 用户确认的品号 |
| `IS_CORRECT` | CHAR(1) | 是否识别正确 (Y/N) |
| `CREATED_TIME` | DATE | 识别时间 |
| `CREATED_BY` | VARCHAR2(50) | 创建人 |

**索引**:
- `IDX_RECOGNITION_PLUNO` - 按品号查询
- `IDX_RECOGNITION_TIME` - 按时间查询
- `IDX_RECOGNITION_CORRECT` - 按正确率统计

## 📊 识别流程

```
用户拍照/选择图片
       ↓
前端调用 /api/product/recognize
       ↓
后端上传图片到 OSS
       ↓
调用阿里云商品识别 API
       ↓
解析识别结果（品号、名称、类别、置信度）
       ↓
返回前端展示
       ↓
用户确认结果
       ↓
记录识别日志（含用户确认信息）
```

## 🔍 故障排查

### 问题 1: 识别返回空结果

**检查**:
1. 日志中是否有 OSS 上传成功信息
2. 阿里云 AccessKey 配置是否正确
3. 网络连接是否正常

### 问题 2: 日志表插入失败

**检查**:
1. 表是否已创建
2. 数据库连接是否正常
3. 字段类型是否匹配

### 问题 3: 识别速度慢

**优化**:
1. 检查 OSS 上传速度
2. 检查阿里云 API 响应时间
3. 考虑异步识别
