## 📝 备注

- 优先级说明：
  - **P0**: 核心基础，必须优先完成
  - **P1**: 核心业务，高优先级
  - **P2**: 重要业务，中优先级
  - **P3**: 辅助功能，低优先级

- API 调用规范：
  - 按照 V3XUI 后端服务格式
  - serviceId 命名方式
  - Authorization header 生成规则

- 每个模块开发前需先确认 V3XUI 中的具体实现逻辑
- 保持与 V3XUI 功能对齐，但采用 Vue3 最佳实践
